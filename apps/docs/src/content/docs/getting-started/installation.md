---
title: Installation
description: Install Minions Content-plans packages
---

## TypeScript / Node.js

```bash
npm install @minions-content-plans/sdk minions-sdk
```

## Python

```bash
pip install minions-content-plans
```

## CLI

```bash
npm install -g @minions-content-plans/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
