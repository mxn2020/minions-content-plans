# Minions Content-plans

Content calendars, themes, and posting schedules per account group

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-content-plans/sdk`) — Core library for Node.js
- **Python SDK** (`minions-content-plans`) — Python bindings
- **CLI Tool** (`@minions-content-plans/cli`) — Command-line interface
- **Web App** — Interactive playground at [content-plans.minions.wtf](https://content-plans.minions.wtf)
